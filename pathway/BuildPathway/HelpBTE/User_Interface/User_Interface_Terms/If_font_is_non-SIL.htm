If font is non-SIL

With the EBook (epub) stylesheet (or a copy) selected in the Stylesheets pane, the Properties tab shows the If font is non-SIL field.

  • For a custom stylesheet, it is available when Embed Fonts in Document is selected ().
    If a source file points to a font that is not distributed by SIL International, you need to tell Pathway what to do. Select one of these options:

    • Embed Font Anyway

    • Use Fallback Font

    • Prompt User

    • Cancel Export.

Important

  • For SIL fonts, you can embed them according to the SIL Open Font License (OFL). See: http://scripts.sil.org/cms/scripts/page.php?item_id=OFL.

  • For non-SIL fonts, you will need permission to embedded them unless the font is in the public domain. For example, fonts supplied with Windows® or Macintosh® cannot be embedded without permission. See, for example, http://msdn.microsoft.com/en-us/library/ms533034(VS.85).aspx.
