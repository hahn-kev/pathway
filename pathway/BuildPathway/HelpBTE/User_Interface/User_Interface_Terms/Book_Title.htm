Book Title

The Book Title field is in the Publication Info tab in both the Set Defaults and Export Through Pathway dialog boxes. It stores the name of the book which is the data in the export file.

  • You need to enter or type the title of the publication manually.

For scripture data, suggested format and content could be:

  • The New Testament in <LanguageName>

The scope could be:

"New Testament" or "New Testament with some Old Testament Portions" or "Bible" or "Scripture Portions", and so on as appropriate.

Uses of the content stored in this field (not exhaustive)

  • In the Front Matter tab:

    • If you select () Include the book's title in the image, it will appear on the cover page.

    • If you select () Title Page (page with Book's title), it will appear on the title page.
