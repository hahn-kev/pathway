Copyright Information Page

The Copyright Information Page check box is in the Front Matter tab in both the Set Defaults and Export Through Pathway dialog boxes. This page is also referred to by other terms including, Verso, Colophon page, Edition Notice page, or Copyright page.

Uses of the check box and button

  • If you select () Copyright Information Page, then a page is included in your export file that displays copyright information:

    • If you select () Standard Right Statement, you can select one of these statements: Draft, Non-Commercial Share Alike, or Intended for Sale.

The path to the file with the standard copyrights statement appears in the Custom box, and statement is included with the export file.

    • If you select Custom, you can click the ellipsis button () and select a customized copy rights statement.

Important

  • The Draft file displays a statement that the document is not in its final form and that it will be superceded.

  • The Non-Commercial Share Alike file refers to http://creativecommons.org/licenses/by-nc-sa/3.0/.

  • The Intended for Sale file only contains the heading "Copyright information". The Copyright Holder symbol and statement are added below that header.
