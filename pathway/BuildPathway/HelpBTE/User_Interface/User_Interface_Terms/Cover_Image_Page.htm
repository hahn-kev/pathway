Cover Image Page

The Cover Image Page check box is in the Front Matter tab in both the Set Defaults and Export Through Pathway dialog boxes. There is an associated Select button.

Uses of the check box and button

  • If you select () Cover Image Page, then a page is included in your export file that displays the image you see in the thumbnail preview. Click the Select button to choose the image.

  • If you select () Include the book's title in the image, the book title you entered appears on the cover page, in the image.

Important

  • The image needs to be less than 1000 pixels in height and length.
