Creator

The Creator field is in the Publication Info tab in both the Set Defaults and Export Through Pathway dialog boxes.

It stores the name of the person, corporation or entity that is responsible for the creation of the data you will export.

  • For scripture data, suggested format and content could be:

 

<Translator'sName> or <TranslationCommitteeName> or Wycliffe Bible Translators, Inc.

Uses of the content stored in this field (not exhaustive)

  • It is included as metadata in export file.

    • For example, if you export to LibreOffice, it appears in the Properties dialog box which you open from LibreOffice when the file is open.
