Red Letter

When Mobile is selected (Scripture only), the Properties tab includes the Red Letter field.

For a custom stylesheet, you can select Yes or No.

  • Yes - allows the application of the Words of Christ style to continue in the export file. You apply this style in FieldWorks Translation Editor or Paratext to specify the red font color. If you have not applied this style, then no words of Christ appear in red, even if Yes is selected.

  • No - blocks (removes) the application of the Words of Christ style in the export file. The default color is used for all content in the export file.

Tip

  • A red letter edition of the Bible displays the words Christ (Jesus) spoke while on earth in red. Words with a red font color can be harder to read on some models of phones.
