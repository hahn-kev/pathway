Base Font Size

When Paper is selected in the Navigation pane, the Properties tab shows the Base Font Size field.

For a custom stylesheet, you can select a font size.

  • You can select point sizes such as 12, 11, 10, 9, or No Change:

  • No Change means that the font sizes output from your content-providing program are not changed.

Important

  • Select a size that is appropriate for the Line Spacing you will use.

    • Standard stylesheets typically set the base font size one, two, or three points smaller than the line spacing. Stylesheets for drafts may specify a larger line spacing, so space is available for hand-written comments.

  • Stylesheets can use your selection to calculate for other font sizes. For example, a dictionary stylesheet could set the font size for homograph numbers to 55% of the base font size.

  • Some publishers have restrictions on font sizes used in large documents. The larger the point size, the more paper it will take to print a publication, so it is important to know their requirements or restrictions.

Tip

  • For EBook, see Font Size.
