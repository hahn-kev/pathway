Copyright Holder

The Copyright Holder field is in the Publication Info tab, which is in both the Set Defaults and Export Through Pathway dialog boxes.

It stores the copyright symbol, organization name, and statement regarding copyright.

  • The selection in the Select Your Organization dialog box populates this field with approved content. You can edit it as necessary.

    • If you use the Organization Name dialog box, must manually edit it.
