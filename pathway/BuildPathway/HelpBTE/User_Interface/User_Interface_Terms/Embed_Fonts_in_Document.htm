Embed Fonts in Document

With the EBook (epub) stylesheet (or a copy) selected in the Stylesheets pane, the Properties tab shows the Embed Font in Document check box.

  • For a custom stylesheet, you can do the following:

    • Select () it to include embedded fonts in the output file.

      In this case, the Also Embed Font Variants check box, and the If font is missing, If font is non-SIL, and Fallback Font fields are also available.

    • Clear () it to not include embedded fonts in the output file.

      In this case, the other check box and fields are not available.

Important

  • For SIL fonts, you can embed them according to the SIL Open Font License (OFL). See: http://scripts.sil.org/cms/scripts/page.php?item_id=OFL.

  • For non-SIL fonts, you will need permission to embedded them unless the font is in the public domain. For example, fonts supplied with Windows® or Macintosh® cannot be embedded without permission. See, for example, http://msdn.microsoft.com/en-us/library/ms533034(VS.85).aspx.
