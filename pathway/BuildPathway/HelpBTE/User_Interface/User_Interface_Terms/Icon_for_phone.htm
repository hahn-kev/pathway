Icon for Phone

When Mobile is selected (Scripture only), the Properties tab includes the Icon for phone field.

The currently selected icon appears next between the label and the Browse button. The icon appears in the mobile device that displays the export file.

  • For a custom stylesheet, you can click Browse to browse for an icon. It must be a *.png or *.icon image file, and be 20 pixels by 20 pixels in size.

Tip

The default icon is installed with Pathway Configuration Tool, typically at one of these locations:

  • C:\Program Files\SIL\FieldWorks\PathwaySupport\GoBible\GoBibleCore\icon\Icon.png

  • C:\Program Files\SIL\FieldWorks 7\PathwaySupport\GoBible\GoBibleCore\icon\Icon.png
