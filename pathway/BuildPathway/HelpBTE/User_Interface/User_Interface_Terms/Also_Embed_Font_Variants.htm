Also Embed Font Variants

With the EBook (epub) stylesheet (or a copy) selected in the Stylesheets pane, the Properties tab shows the Also Embed Font Variants check box. Font variants include bold, italics, small caps and so on.

  • For a custom stylesheet, it is available when Embed Fonts in Document is selected ().
    Then, you can do the following with the Also Embed Font Variants check box:

    • Select () it to include the font variants in the output file.

    • Clear () it to not include the font variants in the output file.

      In this case, the output is smaller, and the EBook reader will make its best attempt to apply the attributes of the variants where they had appeared in the source data. Different EBook readers may have different results.
