Preparation before exporting data

Before you use the Export Through Pathway dialog box, make sure your data is ready. For example (not exhaustive):

  • Sort and filter the data.

  • Configure the dictionary.

  • Make sure picture captions are correct.

  • Make sure examples and example references are current and correct.

  • Configure the reversal index.

  • Generate a fresh Grammar Sketch.

  • Check spellings of words in both vernacular or analysis writing systems.

  • Run any required checks, such as described in theWord/MySword.

  • Apply styles, as desired, to individual writing system, embedded words, and so on.

    • If you will use the FieldWorksStyles or FieldWorksArabicBased stylesheet, make sure the appropriate styles are set in Language Explorer (FLEx). These stylesheet passes most of the style settings you specified in FLEx to the output file. While these stylesheets enable a nearly WYSIWYG output, some adjustments might be needed in the output file. For example, you may need to manually reposition pictures.

Important

  • Pathway is a key way to move lexical data from FieldWorks Language Explorer (FLEx) to a form (LibreOffice, EPUB, and so on) that is closer to becoming publishable, or is actually published.  See: http://fieldworks.sil.org/ and http://pathway.sil.org/.
