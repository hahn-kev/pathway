Export Through Pathway dialog box

  1. Prepare the data.

  2. Open the dialog box, and then do the following:

    • If you will not use the default selections, select a destination and a stylesheet. Then, click More use the Publications Info, Front Matter, and Processing Options tabs.

    • Click the Intellectual Property Info link, and read the information.

    • Select () the "I have complied...." check box.

    • Click OK.

The export operation begins.

  1. If additional dialog boxes or warnings boxes appear, click appropriate options.
    Examples:

    • Font Copyright Warning: Verify that you have sufficient rights or choose another font. Select () or clear () Do this for the next <number> item(s) check box, if it appears.

    • EBook (epub): In Validate .epub File, click Validate to check the output for errors.

For some destinations, that program opens and displays the output (if the program is installed on your computer).

For other destinations, a file is added to the folder specified in Save in Folder (Processing Options tab). You need to open those files with the appropriate program.

Note

  • If you will export more than one reversal index, export them separately. Then, merge them in a common file. Only the displayed reversal index is exported.

  • Processing a dictionary and Processing Scripture provide additional information.

  • You can click the button () to use the Select Layout dialog box.
