Front Matter tab

Do the following in this tab:

  1. To include a cover image page, select () Cover Image Page.

    • Click Select to select a different image.

    • If you want the book title to appear in the image, select () Include the book's title in the image.

  1. To include a title page, select () Title Page.

  2. To include a copyright information page, select () Copyright Information Page. Then, do either for the following:

    • Select Standard Rights Statement. Then, click the down arrow and select a rights statement.

    • Select Custom. Then, click the ellipsis button () and select your custom *.xhtml file.

      Click Help me choose a rights statement for assistance.

    1. To include a table of contents, select () Table of Contents.

    2. Click the Processing Options tab.

Note

  • This tab is not available for use for these destinations:

    • theWord/MySword. The metadata is hard-coded in exported files. If you need to edit them, request technical support.

    • DictionaryForMids as front matter is not supported

    • PDF (using Prince).

  • Both the Set Defaults and Export Through Pathway dialog boxes have this tab.
