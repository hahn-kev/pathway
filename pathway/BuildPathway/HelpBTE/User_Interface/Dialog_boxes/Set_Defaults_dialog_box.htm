Set Defaults dialog box

This dialog box has a different title depending on the selection in the Navigation pane: Set Defaults - Dictionary or Set Defaults - Scripture. You need to set the defaults for dictionary and scripture data separately.

The selections you make in this dialog box are the initial (default) selections and content in the Export Through Pathway dialog box. Some can become RAMP Metadata.

  1. To open this dialog box, click the Defaults toolbar button, or click OK in the Select Your Organization dialog box.

  2. Select a destination.

    • Click to open the Destination Help topic (optional).

  3. Select a stylesheet.

    • Use the Select Layout dialog box (optional).

  4. Click the More button to display and use these tabs:

    • Publication Info

    • Front Matter

    • Processing Options.

  5. Click the Intellectual Property Info link, read that information.

    • Select () the check box to indicate compliance with your organizations policies.

  6. Click OK.

Note

  • The Stylesheet control is not available for use if the destination is theWord/MySword. The stylesheet control comes from the content-providing program, such as Paratext. If you need to change them, request technical support.
