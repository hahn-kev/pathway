Publication Info tab

  1. Do the following in this tab:

    • Book Title - type the name of the publication.

    • Description - type a description of the publication.

    • Creator - type the name of the person, group, organization, or other entity who created this publication.

    • Publisher - type the name of the publisher.

    • Copyright Holder - update or enter new information, if what is displayed is not already complete or correct. You need to do this if you used the Organization Name dialog box to enter a new name.

  2. Click the Front Matter tab.

Important

  • Both the Set Defaults and Export Through Pathway dialog boxes have this tab.

  • Some metadata in this tab are automatically set according to your selection in the Select Your Organization dialog box. Initially, the Publisher and Copyright Holder fields are automatically populated, at least partially.

    • Various *.xml file determines this.

Each organization can specify default metadata that will appears in other fields, depending on how extensively they want to provide this metadata. This metadata appear in various locations in programs that use the data, such as in theWord.

  • For more information, request technical support.
