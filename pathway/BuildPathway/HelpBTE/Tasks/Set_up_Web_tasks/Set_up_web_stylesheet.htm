Set up web stylesheet

Prerequisites

  • You or someone must set up a website with a web hosting service.

  • You must have the details from the hosting service available to enter into the Properties tab in Pathway.

To see the appropriate Properties tab, select Web in the Navigation pane, and then select the desired stylesheet in the StyleSheets pane. Then, do the following:

FTP Details

  • Enter the FTP address, user name, and password.

Source: the hosting service.

MySql Database Details

  • Enter the MySql server name/IP address, Database name, and user name and password.

Source: the hosting service.

Website Details

  • Enter the following details that are specific to the particular site where you will send your data:

    • Website url: the web address, or path to the website, which includes the domain name.

    • Admin Username and Admin Password: You are the administrator if you set up the web hosting service.

    • Site Title: This a brief text that you want to appear at the top of the web pages to identify the site contents. For example, you could type a language name followed by "Dictionary", in the language of the intended readers.

    • Email ID: This is the email address of the person who requested the website from the providing hosting service.

    • Ftp Folder Name: This name will be added to the end of the website url. It is the name after the last forwardslash (/).
