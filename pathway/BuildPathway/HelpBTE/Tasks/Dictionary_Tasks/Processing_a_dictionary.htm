Processing a dictionary

Language styles gives the general steps to prepare the data you will export.

Here are some additional considerations (not exhaustive):

  • If you will use the FieldWorksStyles stylesheet in Pathway, make sure the appropriate styles are set in FieldWorks Language Explorer (FLEx). (This stylesheet passes most of the style settings you specified in FLEx to the output file.)

    • While this stylesheet enables a nearly WYSIWYG output, some style adjustments might be needed in the output file. For example, you may need to manually reposition pictures.

  • The standard stylesheet named FieldWorksArabicBased is for a Arabic right-to-left writing systems.

  • For more information, such as presenting data on an Internet site ( Web), see Set up Web overview and request technical help.
