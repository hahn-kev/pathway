Create a custom rights statement

You can choose from several standard rights statements. If none of them are appropriate or to localize one into another language, create a custom statement.

  1. Open the Copyrights folder which is typically located here:

    • Windows: C:\Program Files\SIL\Pathway7\ or C:\Program Files (x86)\SIL\Pathway7\.

    • Linux: /usr/lib/pathway/.

  2. Copy the desired *.xhtml file, such as Custom_Template.xhtml.

    • Paste it into a folder that is outside of Program Files, such as one in My Documents (Windows) or outside of usr, such as one in Home (Linux).

    • Rename the file, but keep the .xhtml filename extension.

  3. Open that file in a text editor such as Notepad or ZEdit.

    Do not use Word to edit the file. That will create extraneous markup in the .xhtml file that can cause unexpected results.

    • Make the necessary edits to the statement, typically between these lines:

      <div id='LicenseInformation' class='Front_Matter'>

           and

      </div>.

    • Save your work.

    • Open the file in an Internet browser to verify your content and formatting.

  4. Have your statement reviewed by a legal counsel.

  5. Choose that custom rights statement in the Front Matter tab.

Note

  • If you store your a custom rights statement in the Copyrights folder it will be lost when you get a newer version of Pathway.

  • Request technical support if necessary.

  • Regarding the text direction (left-to-right, or right-to-left), the text direction is set in the license or statement file, not in Pathway.

    • If you create a custom statement or license for a majority/trade language that is right-to-left, you need to set that text direction in the xhtml file.

    • For Creative Commons (CC) licenses, some of this localization is already done. There are affiliate entities in a number of countries: http://wiki.creativecommons.org/CC_Affiliate_Network. Each affiliate typically has a set of license agreements that are legally binding in their area. Possibly, individual areas of SIL, or partner organizations, could create a license that meshes the CC license agreement in a custom license agreement.
