Setting default selections

The selections you make in the dialog boxes below are the initial selections (the defaults) that appear in the equivalent dialog boxes (more) when you export data from a content-providing program.

Some can become RAMP Metadata.

  1. Display the appropriate list of stylesheets.

  2. On the toolbar, click Defaults, or press the shortcut keys Alt +F.

    The Select Your Organization dialog box appears if you have not previously selected your organization. If you have, then the Set Defaults dialog box appears.

  3. Use the features in the Select Your Organization dialog box, and then click OK.

    The Set Defaults dialog box appears.

  4. Use the features in the Set Defaults dialog box, and then click OK.
