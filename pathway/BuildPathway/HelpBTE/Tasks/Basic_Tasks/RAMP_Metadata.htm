RAMP Metadata

This table shows how Pathway sets some of the RAMP metadata when you archive data. In RAMP, you will want to verify the metadata. Some will appear in English by design, and other is intentially multi-lingual or language specific.

You will find more lines in RAMP. Some are required (e.g. | Work Stage); others are optional (e.g. Additional Titles) or recommended (e.g. Contributors).

Wider Audience

Pathway selects Text.

Pathway selects Monograph.

  • Y (to indicate it is concerned with a particular language)

Content Language and Script

All current vernacular, pronunciation,and analysis writing systems

Contributors

Creator field in the Publication Info tab (Set Defaults dialog box), and Pathway selects compiler.

Date

The date you exported the data through Pathway.

Description

Pathway selects that there is a general description (Y), and inserts content from the Description field in the Publication Info tab (Set Defaults dialog box), and the language of the description.

Foreign Literatures and Linguistics; Language Documentation; Dictionary; Reference, English.

Software Requirements

Font you used, and possibly software and schema information.

Rights

  • Copyright Holder field in the Publication Info tab (Set Defaults dialog box)

  • Rights Description from selection in Front Matter tab (Set Defaults dialog box).
