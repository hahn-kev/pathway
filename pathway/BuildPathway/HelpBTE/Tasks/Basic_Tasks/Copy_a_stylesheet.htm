Copy a stylesheet

  1. Display the appropriate list of stylesheets.

  2. In the Stylesheets pane, click the stylesheet you want to copy.

  3. On the toolbar, click Save As, or press the shortcut keys Alt+A.

    The selected stylesheet is copied as a custom stylesheet.

  4. Click the Info tab, and then do the following:

    • Edit the name of the stylesheet.

    • Type the description of the stylesheet.

    • If you want to make the stylesheet available, select () Available check box.

    • Type a comment (optional).

  5. Click the Properties tab, and then do the following:

    • Type a new value or make a different selection in each field that you need to change.
