Choosing a rights statement - Other Organizations

Pathway has added several license agreements which you can add to your document. You select them in the Standard Rights Statement field, which is in Front Matter tab of the Set Defaults or Export Through Pathway dialog boxes.

The following logic flow should help you decide what license agreement is appropriate for your situation.

If the publication

Select this Standard Rights option

is a draft document intended for review or village testing

Standard Rights Statement

Intended for Sale

will be shared and can be used to create a derived publication (for example, dictionary data being used for a language primer)

Non-Commercial Share Alike

will be shared, but no derived publications are allowed.

Non-Commercial No Derivatives

Important

  • The Draft file displays a statement that the document is not in its final form and that it will be superceded.

  • The Non-Commercial Share Alike file refers to http://creativecommons.org/licenses/by-nc-sa/3.0/.

  • The Intended for Sale file only contains the heading "Copyright information". The Copyright Holder symbol and statement are added below that header.

 Note

  • If none of these are appropriate for your situation, create a custom rights statement.

  • If you will publish data using Pathway that has already been published, contact the original publishing entity to make sure the publication through Pathway does not violate existing publishing or license agreements.
