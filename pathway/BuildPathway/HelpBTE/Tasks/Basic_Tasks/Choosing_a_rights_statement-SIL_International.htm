Choosing a rights statement - SIL International

In the Front Matter tab in the Set Defaults and Export Through Pathway dialog boxes, you can choose a standard rights statement, or select a custom rights statement.

  • For academic and vernacular publications were SIL holds rights:

    • If it is going to be sold, then use: © YYYY, SIL International. All rights reserved.

    • If it is going to be shared and can be used to create a derived publication, then use: © YYYY, SIL International. Redistributable under a CC: BY-NC-SA

    • If it is going to be shared but no derived publications are allowed, then use: © YYYY, SIL International. Redistributable under a CC:BY-NC-ND

  • For Scripture publications were SIL holds rights:

    • If Scripture is going to be sold when redistributed, then use: © YYYY, Wycliffe Bible Translators, Inc. (or another appropriate publisher from a publishing agreement such as Bible League International, a national Bible Society, etc.)

    • If Scripture is going to be shared no derived publications are rarely allowed, so use: © YYYY, Wycliffe Bible Translators, Inc. Redistributable under a CC:BY-NC-ND.

Note

  • Sometimes Scripture is shared in order to create derived publications. For those situations we would usually do a one-time agreement.

  • The proposed iterative publication processes discussed in Chiang Mai at the Publishing Technical Summit would need a living license or controlled distribution. Nothing like this system or licensing currently exists.

    • One possibility: © YYYY, Wycliffe Bible Translators, Inc. Redistribution of Working Drafts on active Translation Projects allowed under a modified CC:BY-NC-SA*.

  • If you will publish data using Pathway that has already been published, contact the original publishing entity to make sure the publication through Pathway does not violate existing publishing or license agreements.
