Specifying metadata

  1. Open the Set Defaults (or the Export Through Pathway) dialog box.

    • Select the desired destination and stylesheet, and then click the More button.

    • Make appropriate selections or type metadata in the fields in these available tabs.

  2. For stylesheet metadata, refer to the Stylesheet pane.
