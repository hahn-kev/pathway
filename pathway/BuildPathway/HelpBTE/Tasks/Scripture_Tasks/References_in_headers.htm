References in headers

For custom stylesheets, you can select a position for the running header references in the page headers of the output file, and then you can select the reference format. These work together with page number options.

  1. Display the appropriate list of stylesheets.

  2. In the Stylesheets pane, click the desired custom stylesheet.

  3. In the Stylesheet pane, click the Properties tab.

  4. In the Running Header field, select Mirrored or Every Page.

  5. In the Reference Format field, select one of the available reference format options.
