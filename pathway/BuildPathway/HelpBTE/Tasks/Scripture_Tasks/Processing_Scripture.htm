Processing Scripture

The Help topic for the Export Through Pathway dialog box gives the general steps to prepare the data you will export.

Here are some additional considerations (not exhaustive):

  • It does not matter which view is selected in the FieldWorks Translation Editor (TE) Navigation pane. That is, the output result is the same if you export from Scripture or from another view.

    • Exception: If you display the vernacular Scripture in the TE Trial Publication view, it appears in two columns. Then, if you select a Pathway stylesheet that specifies two columns, the columns may not wrap properly at section breaks.

    • If you select the Parallel Print Layout view in Back Translation, only the vernacular Scripture text, but not the back translation.

    • You need to filter by book. All currently displayed books are exported.

  • If you export Scripture and select Go Bible, any style formatting is ignored. Java files (*.jar) are generated. The output is then only usable by Java-enabled phones.

  • C5 Cols ApplicationStyles stylesheet: If you  use this stylesheet, make sure the appropriate styles are set in Translation Editor or Paratext. This stylesheet passes most of the style settings you specified in the scripture editor to the output file.

    • While this stylesheet enables a nearly WYSIWYG output, some style adjustments might be needed in the output file. For example, the first verse of each chapter may appear differently than the other verses.
