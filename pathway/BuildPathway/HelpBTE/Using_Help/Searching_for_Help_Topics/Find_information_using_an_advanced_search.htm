Find information using an advanced search

  1. Click the Search tab.

  2. Type the words or phrases you want to find in the Type in the word(s) to search for list.

  3. Use any nested expressions for complex searches.

  4. Click    to add boolean operators to your search terms.

  5. Click List Topics.

  6. To sort the topic list, click the Title, Location, or Rank column heading.

  7. Select the topic you want, and then click Display.

  8. Click the topic you want.

 Note

  • You can also:

    • Find words similar to your search term.

    • Search only the last group of topics.

    • Search topic titles only.
