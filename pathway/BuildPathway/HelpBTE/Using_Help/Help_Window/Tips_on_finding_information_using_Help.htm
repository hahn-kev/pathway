Tips on finding information using Help

Here are some tips on how to find more information when using the Help window:

  • To link to another topic, a Web page, a list of other topics, or a program, click the blue colored and blue underlined text.

  • To view topics that contain related information, click a link under Related Topics, which appears at the end of each topic.

  • To see if a word or phrase contained in a topic is in the index, select the word, and then press F1.

  • If you are viewing content from the Web in the Topic pane, you can click Stop or Refresh on the toolbar to interrupt a download or refresh a Web page.

  • If you use a particular help topic often, you can add it to your favorite topics.

  • Right-click in the Content or Topic pane, and then select Print to print the current help topic.

 Note

  • You can also find topics using the toolbar buttons.
