Hide or show the Navigation pane

  • On the toolbar, click the Close button Hide or Show to close or display the Help Navigation pane, which contains the Contents, Index, and Search, and Favorites tabs.

  • To display the Help Navigation pane, click the Contents, Index, or Search tab.

 Note

  • If you close the Help window with the Help Navigation pane hidden, it may appear that way when you open the Help window again.
