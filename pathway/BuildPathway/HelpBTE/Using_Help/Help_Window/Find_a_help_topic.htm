Find a Help topic

In the Navigation pane, click one of the following tabs:

  • To browse through a table of contents, click the Contents tab. The table of contents is an expandable list of all topics.

  • To see a list of index entries, click the Index tab, and then type a word or scroll through the list. Double-click an index word.Click an index word.

    • If only one Help topic is indexed under the keyword, then the Help topic will appear in the Topic pane.

    • If two or more Help topics are indexed under same keyword, the Topics Found window will appear. Select a topic from the list, and then click the Display button.

    • If two or more Help topics are indexed under the same keyword, a list of topics will appear. Select a topic from the list, and then click.

  • To find every occurrence of a word or phrase contained in all the Help files, click the Search tab. Type the word in the Type in the word(s) to search for list, click List Topics, and then double-click a topic and then click a topic.

 Note

  • Search for Information overview explains, in detail, how to use the Search tab.
