Understanding Help window panes

The Help window contains a toolbar and two panes.

  • The Help toolbar is below the Using Help window title barInternet Explorer toolbars.

  • The Navigation pane is on the left of the window. It contains four tabs: the Contents tab, the Index tab, the Search tab, and the Favorites tab.

  • The Navigation pane is on the left of the window. It contains three tabs: the Contents tab, the Index tab, and the Search tab.

  • The Topic pane (also known as the Content pane) is on the right of the window. It displays the selected Help topic.

 Note

  • Understanding these panes will help you find a help topic.
