Print the current Help topic

Do one of the following:

  • Click Print on the Help toolbar.

  • Click the Print button in the Help Navigation pane.

  • Right-click in the Topic pane, and then click Print.

  • Right-click a topic on the Contents tab, and then click Print.

 Note

  • You can choose to print the selected topic or all topics in the selected heading (that is, all the topics in a book).
