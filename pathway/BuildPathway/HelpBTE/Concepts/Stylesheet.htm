Stylesheet

A stylesheet (or style sheet) is a document that consists of settings that describe the layout and presentation to be used when displaying other documents.

Stylesheets enable you to separate document content from document presentation because they store the specification of presentation elements such as the layout, colors, and fonts.

In the Pathway Configuration Tool, the set of standard style sheets are built up from a series of snippets. These cascade to give a predictable result.
