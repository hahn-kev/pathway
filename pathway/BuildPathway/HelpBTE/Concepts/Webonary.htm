Webonary

Webonary is a server-side application, that is, it is software. It is used to publish to a web hosting services of your choice. This is not the same as SIL's Webonary.org hosting service, which is the preferred web publishing option.

See: http://webonary.org/

Use Webonary.org to host your data

  • In this case, you do not use Pathway. Instead you export the dictionary data directly from FieldWorks Language Explorer as an XHTML file. Then, you import it into the Webonary site you set up. See: http://webonary.org/wp-signup.php/ or http://pathway.sil.org/webonary/webonary/host-your-dictionary-at-webonary-org/.

Choose a different web hosting service

  • After you have chosen and set up a hosting service, and set up a web stylesheet in the Pathway Configuration Tool, then a destination called Webonary is available in the Set Defaults and Export through Pathway dialog boxes.

In this case, you export dictionary data from FieldWorks Language Explorer through Pathway, using your web stylesheet and metadata.

Note

  • Set up Web overview discusses how the size of the dictionary may determine which site and process you use.
