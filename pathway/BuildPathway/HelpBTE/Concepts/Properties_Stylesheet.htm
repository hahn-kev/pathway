Pathway Style Sheet Properties

The Pathway Configuration Tool (PCT) project prepares FieldWorks or Paratext XHTML export (and eventually LIFT and OXES formatted data) for printing using LibreOffice or other rendering engines. LibreOffice works very much like Microsoft Office 2000/2003 program, but supports rendering complex fonts using Graphite Technology and fonts. The style sheet for the conversion of data for printing is based on the properties of the Cascading Style Sheet 2.1 standard. The supported properties are given in the table below. (Property names in parenthesis are not implemented.) FieldWorks or Paratext XHTML export will create a *.CSS file with these properties already included for the default formatting of the lexical data. The properties given below can be used to change the rendering of any style tag in the exported file.

At the end of the first table is the Custom Properties table. It lists properties that were added to help support the conversion and formatting of lexical and Scripture data.
