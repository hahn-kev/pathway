Cascade

Stylesheets store the specification of document presentation elements such as the layout, colors, and fonts. Cascading is the process of combining style sheets and features so the end result appears to be a single new style sheet. As style sheets are cascaded, properties in a particular context over-ride the same property in the same context given earlier.

Said another way, cascading allows the use of multiple snippets or stylesheets in a particular order (or with a priority scheme) to determine which specification applies if more than one matches a particular element. Then, the results are predictable.
