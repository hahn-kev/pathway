XeLaTex

XeTeX a typesetting system based on a merger of Donald Knuth's TeX system with Unicode and modern font technologies. It is a command-line tool, like standard TeX and pdfTeX processors. LaTeX is a macro package which provides a set of macros for TeX to interpret. This format generates a pdf file allowing technical users precise typological control over the output.

If installed, XeLaTex is a Pathway destination you can choose to generate pdf files, using the selected stylesheet.
