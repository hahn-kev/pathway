OpenOffice and LibreOffice

OpenOffice and LibreOffice use the OpenDocument format (http://en.wikipedia.org/wiki/OpenDocument). This is an open XML-based word processing format that can be read by many word processing applications. This format retains text styling as well as font and pictures. Additional editing can be done in this format before final publication.

 Note

  • Of these applications, LibreOffice, OpenOffice, and KOffice use the OpenDocument format as their native file format.

  • After installing LibreOffice or OpenOffice, on the Tools menu, point to Options to Security then to Macro Security to set the security level to Medium.

  • Currently, if you want Microsoft Word as your destination, you need to export to OpenDocument, and then either open it with a compatible version of Microsoft Word or export it from an OpenDocument-compatible word processor into a Word format.
