Dublin Core

From http://dublincore.org/:

  • The Dublin Core Metadata Initiative, or "DCMI", is an open organization engaged in the development of interoperable metadata standards that support a broad range of purposes and business models.

Along with the exported dictionary or scripture data, Pathway exports metadata that is consistent with DCMI. Some of these metadata elements are automatically generated by Pathway, while others can be configured in the Publication Info tab on the Set Defaults or Export Through Pathway dialog box.
