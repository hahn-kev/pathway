theWord/MySword

In the Set Defaults-Scripture and Export Through Pathway dialog boxes the Destination control allows you to choose theWord/MySword.

This destination exports files for both theWord (a computer program) and also MySword (for Androidâ„¢ mobile devices). Output filename extensions include: *.nt, *.ot, *.ont, and *.bbl.mybible.

See Also: https://github.com/sillsdev/pathway/wiki/theWordOnt.

Important

Before you do the export, run basic checks for (minimally) Chapter/Verse Numbers, Markers and References in Paratext and make sure there are no errors. Otherwise, you could experience problems during the export process.

Choices

If theWord program is installed when you export scripture, you are asked if you want to start theWord.

  • If you click Yes, the files are programmatically copied to the correct folder to be available to theWord. Then, theWord programs opens.

  • If you click No, the folder opens. The folder contains both the theWord file and the MySword files. In this case you can copy them to the desired location manually. Also you can send the MySword file to pathway@sil.org for uploading.

  • If you click Cancel, nothing further happens. The export is already finished and the files are in the folder, but nothing else happens programmatically.

File name extensions

  • For theWord, you may see three different filename extensions:

    • <name>.nt files contain only New Testament data

    • <name>.ot files contain only Old Testament data

    • <name>.ont files contain both.

  • MySword files are <name>.bbl.mybible
