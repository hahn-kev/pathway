Destination

These Destination options are available in the Set Defaults and Export Through Pathway dialog boxes. In some cases, they are only available if the associated program is installed on your computer.

Destination

Description

DictionaryForMIDs

DictionaryForMIDs is for Android™, Java Micro Edition, Java Standard Edition and htmlApp devices. This format does not support hyperlinks. The Front Matter tab is not available as front mater is not supported.

E-Book (.epub)

E-Book (.epub) is an open electronic book reader format that can be read by computers, smart phones, tablets and dedicated e-book reader software, such as calibre.

Go Bible

Go Bible is a micro-Bible reader for Java Micro Edition-enabled phones.

InDesign

Adobe®InDesign® is used to design and preview page layouts for print or digital distribution.

OpenOffice/

LibreOffice

OpenOffice and LibreOffice use the OpenDocument format. This is an open XML-based word processing format that can be read by many word processing applications. This format retains text styling as well as font and pictures. Additional editing can be done in this format before final publication.

Pdf (Using Open Office/LibreOffice)

Your data is exported to OpenOffice or LibreOffice, and then uses the program to convert its output to *.pdf. In this case, additional editing can not be done in this format before final publication, however the rights statement is attached to the pdf file.

Pdf using Prince

Prince converts documents to *.pdf. Currently, for lexical data, you need to generate the main dictionary and each reversal index separately. The Front Matter tab is not available and no front matter content is included.

theWord/

MySword

This destination exports files for both theWord (a computer program) and also MySword (for Android™ mobile devices). Output filename extensions include: *.nt, *.ot, *.ont, and *.bbl.mybible. The Front Matter tab is not available.

Webonary

Webonary is a server-side application, that is, it is software. It is used to publish to a web hosting services of your choice. This is not the same as SIL's Webonary.org hosting service, which is the preferred web publishing option.

XeLaTex

XeTeX a typesetting system based on a merger of Donald Knuth's TeX system with Unicode and modern font technologies. It is a command-line tool, like standard TeX and pdfTeX processors. LaTeX is a macro package which provides a set of macros for TeX to interpret. This format generates a pdf file allowing technical users precise typological control over the output.

Important

  • Archive data through Pathway - With each of the output options listed above, Pathway generates a <name>.ramp file which is a RAMP package. You can import this file into RAMP (SIL only).
