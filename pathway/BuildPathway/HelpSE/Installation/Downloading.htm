Downloading

The Pathway SE (Standard Edition) has stylesheets for use with Configured Dictionary data and Reversal Index data.

  1. Remove (uninstall) any previous version from your computer.

  2. Go to the Internet site listed below, and then do the following:

    • In the Search box, select Current Downloads, and then type .msi in the For box.

    • Click the appropriate setup file that has the latest date (in the filename) or most recent Uploaded time.

    • In the dialog box that appears, click Run or Save.

    • If your computer still has a version of the tool installed, click Remove to remove the program. Then, return to the Internet site and repeat the steps given above.

    • Follow the instructions in the installer.
