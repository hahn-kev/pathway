Search for Information overview

A search consists of the words or phrases you want to find. How you represent these words or phrases is called a search term. You may use the Search tab to:

  • Find information using a simple search.

  • Find information using an advanced search.

The basic rules for searching are as follows:

  • Searches are not case-sensitive, so you can type your search in uppercase or lowercase characters.

  • You can search for any combination of letters (a-z) and numbers (0-9).

  • Punctuation marks (for example, the period, colon, semicolon, comma, and hyphen) are ignored during a search.

  • You can use double quotes or parentheses to group the elements of your search. You cannot search for quotation marks themselves.

 Note

  • To search for a file name with an extension, group the entire string in double quotes (for example, "filename.ext"). Otherwise, the period will break the file name into two separate words. The default operation between words is AND, so you will create the logical equivalent to filename AND ext.
