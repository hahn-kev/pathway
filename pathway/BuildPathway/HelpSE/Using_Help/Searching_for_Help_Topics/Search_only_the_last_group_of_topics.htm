Search only the last group of topics

This feature enables you to narrow a search that results in too many topics found. You can search through your results list from previous search by using this option.

  1. Click the Search tab.

  2. Type the words or phrases you want to find in the Type in the word(s) to search for list.

  3. Select the Search previous results check box.

  4. Click List Topics.

  5. Click the topic you want, and then click Display.

 Note

  • If you want to search through all of the Help topics, you must clear the check box.

  • The Search tab will open with this check box selected if you previously used this feature.
