Shortcuts for the Options menu

To

Press

Display the Options menu on the Help toolbar.

Alt+O

Hide or show the Navigation pane.

Alt+O, and then press T

Move back to the previous topic.

Alt+O, and then press B

Return to the home page (help authors can specify a home page for a help system).

Alt+O, and then press H

Stop the Help window from opening a page (this is also useful if you are linking to the Web and want to stop a page from downloading).

Alt+O, and then press S

Refresh the topic that appears in the Topic pane (this is useful if you have linked to a Web page).

Alt+O, and then press R

Change Microsoft Internet Explorer settings. The Internet Options dialog box contains accessibility settings. To change these settings click the General tab, and then click Accessibility.

Alt+O, and then press I

Print a topic.

Alt+O, and then press P
