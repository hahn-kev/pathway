Work with topics using the toolbar buttons

There are several buttons on the toolbar in the Help window. You can click these buttons to work with Help topics:

  • Hide/Show hides or shows the Help Navigation pane on the left.

  • Hide hides the Help Navigation pane on the left.

  • Back displays the previous topic that you viewed.

  • Forward displays the next topic in a previously displayed sequence of topics.

  • Refresh reloads the selected topic.

  • Home displays the Home page topic for the Help file you are viewing.

  • Print prints the Topic pane only

 Note

  • The Help toolbar in your Help window may not contain all of these buttons or may have additional custom buttons.
