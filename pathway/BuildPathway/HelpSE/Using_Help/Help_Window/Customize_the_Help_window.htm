Customize the Help window

There are a few ways to easily change the size, position, and the panes of the Help window:

  • To resize the Navigation or Topic pane, point to the divider between the two panes. When the pointer changes to a double-headed arrow, drag the pointer right or left.

  • To proportionately shrink or enlarge the entire Help window, point to any corner of the Help window. When the pointer changes to a double-headed arrow, drag the pointer to resize the Help window.

  • To change the height or width of the Help window, point to the top, bottom, left, or right edge of the Help window. When the pointer becomes a double-headed arrow, drag the pointer to adjust the height or width of the window.

  • To reposition the Help window on your screen, click the title bar and drag the window to a new position.

You can also:

  • Hide or show the Navigation pane.

  • Change the font size of a topic.

  • Change the formatting or style of a topic.

 Note

  • The Help window will appear with the last size and position settings specified.
