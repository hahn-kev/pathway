Change the font size of a topic

  1. Start Internet Explorer.

  2. On the View menu, click Text Size.

  3. Select your choice of text size (Medium is the default).

 Notes

  • If a Help window is open, close it, and then open it again to see the change in font size.

  • The font size does not apply to the Navigation pane or Help toolbar.

  • This will also change the font size in Internet Explorer.
