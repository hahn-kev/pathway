Copy a Help topic

  1. Navigate to the Help topic you want, and display it.

  2. Right-click in the Topic pane, and then click Select All.

  3. Right-click again, and then click Copy.

    This copies the topic to the Clipboard.

  4. Open the document to which you want to copy.

  5. Move the insertion point to where you want the information to appear.

  6. On the Edit menu, click Paste.

 Note

  • If you choose Select All, you can copy some extra-unintended graphics and miscellaneous characters. You might want to delete these graphics and characters after you paste.

  • If you want to copy only part of a Help topic, select the text you want to copy, right-click, and then click Copy.
