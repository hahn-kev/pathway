Change the formatting or style of a topic

  1. On the OptionsTools menu, click Internet Options.

  2. In the Internet Options dialog box, click Accessibility.

  3. In the Accessibility dialog box, select the options you want.

  4. Click OK two times.

 Note

  • These changes do not apply to the Navigation pane or Help toolbar.

  • This will also change your accessibility settings for Internet Explorer.
