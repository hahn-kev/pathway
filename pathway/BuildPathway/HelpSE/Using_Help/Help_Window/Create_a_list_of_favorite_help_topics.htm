Create a list of favorite Help topics

  1. Locate the Help topicWeb page you want to mark as a favorite.

  2. Click the Favorites tab. On the Favorites menu, click Add to Favorites.

    The Add Favorite dialog box appears.

  3. Click Add. Click OK.

 Note

  • You can put a favorite Web page in a created subfolder, if desired.

  • To return to a favorite Web page: on the Favorites menu, click the Web page.

  • To return to a favorite topic: click the Favorites tab, select the topic, and then click Display.

  • To rename a Web page: on the Favorites menu, right-click the Web page, click Rename, type a new name in the box, and then click OK.

  • To rename a favorite topic: click the topic, type a new name in the box, and then press Enter.

  • To remove a favorite Web page: on the Favorites menu, right-click the Web page, click Delete, and then click Yes.

  • To remove a favorite topic: select the topic, and then click Remove.
