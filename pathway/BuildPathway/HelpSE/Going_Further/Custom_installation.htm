Custom installation

As an advanced task for experienced individuals, a customized installation for a particular location can be made. Request technical support before you attempt this.

Currently, there are two settings files:

  • StyleSettings.xml is used when the program first starts. It keeps track of the font for the user interface and the last output selections used in the Navigation pane.

  • DictionaryStyleSettings.xml has the settings for working with Dictionary stylesheets.

These files are considered to be read-only files because they are installed by the installer, and changing them will cause problems when installing future versions of the program. Consequently, if you do change one of these files, the changed version needs to be saved appropriately in

  • C:\Documents and Settings\All Users\Application Data\SIL\Pathway (XP)

  • C:\ProgramData (Vista, or Windows 7/8).
