File management

Each time you use a stylesheet from a content-providing program, a set of folders and files is created on your hard drive.

WindowsÂ® OS

The default folder structure looks like this:

  • The My Documents\Publications folder has a folder that has the same name as the language project you published.

    • That folder contains a Dictionary folder.

      • That folder contains a folder that has the same name as the stylesheet you used, plus the date and time.

        • In each of these folders there is a copy of the output file and a <name>.ramp file (See: Archive data through Pathway).

Example

From SIL FieldWorks, a Sena 3 dictionary was published to Open Office.

Linux OS

The default folder is located at: Home\<username>\Publications.

Tip

  • You can rename <name>.ramp file to <name>.zip, and then extract the other files (stylesheet, mets.xml, and other files).

  • You can choose a different Save in folder in the Processing Options tab.
