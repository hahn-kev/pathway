About Pathway

Open the About Pathway dialog box using Help on the toolbar.

This dialog box contains:

  • The version number and build date.

Provide this information if you request technical support.

  • License information (http://www.gnu.org/licenses/gpl.html).

  • Copyright and provider information.

  • A link to http://pathway.sil.org/.

  • The Help Improve check box.

If selected (), Pathway usage data is reported to the development team.
