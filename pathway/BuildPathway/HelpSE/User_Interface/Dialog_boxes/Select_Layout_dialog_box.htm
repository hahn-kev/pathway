Select Layout dialog box

In the Set Defaults and the Export Through Pathway dialog boxes, a button () is next to the Stylesheet button. It opens the Select Layout dialog box so you can see a list of available stylesheets, and a preview pane that show sample data for each standard stylesheet.

Do the following in the Select Layout dialog box:

  1. In the left pane, click a stylesheet.

  2. In the right pane, examine the layout of the sample data.

    • Click next () or previous () to see even and odd numbered pages.

  3. Repeat the steps above until the stylesheet you want is selected, and then click OK.

    This dialog box closes, and the stylesheet you selected appears in the Stylesheet button of the open dialog box.

Note

  • If you opened the Select Layout dialog box from the Export Through Pathway dialog box, you can click Edit Layout button (bottom left corner). This opens the Pathway Configuration Tool.
