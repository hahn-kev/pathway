Dialog boxes overview

Here are the dialog boxes that can appear with you use Pathway:

  • Select Your Organization

    • Organization Name

  • Set Defaults / Export Through Pathway

    • Publication Info tab

    • Front Matter tab

    • Processing Options tab

  • Select Layout

Important

  • Set Defaults and Export Through Pathway are two instances of the same dialog box:

    • Set Defaults opens from the Pathway Configuration Tool.

    • Export Through Pathway opens from content-providing programs.
