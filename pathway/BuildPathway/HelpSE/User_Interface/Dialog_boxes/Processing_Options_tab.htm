Processing Options tab

As desired, or as necessary, do any of the following in this tab:

  1. Click the ellipses button () for the Save In Folder field, and then select the folder for the output file.

  2. Select () Replace styles with direct formatting in OpenOffice/LibreOffice if your output file is large. See: OpenOffice/LibreOffice Issues.

  3. Select () the lexical data you want to include in the file:

    • Configured Dictionary

    • Reversal Indexes.

  4. In the Preprocessing Transformation box, select () any of the following:

    • Filter Broken Links removes hyperlinks that are broken because the entries that they point to are not in the output file. Filters in the content-providing program could cause this.

    • Filter Empty Entries filters out potentially empty lexical entries. Filters and configuration settings in the content-providing program could result in entries that are headwords without any content.

    • Fix Duplicate ids solve validation errors caused by lexical items that have the same internal identification numbers.

    • FixLinksBeginWithSpace retains the hyperlink properties even if the link begins with an empty space.

    • Remove Letter Dividers or Remove Letter Dividers2 removes the letters that separate the sets of words that start with different first letters (one column and two column stylesheets, respectively).
      Remove unwanted Letter Dividers       attempts to prevent characters that are secondary sort characters, such as tone marks, from becoming letter dividers.

    • Remove Space Before Picture Entry removes the unwanted space that may appear just before the first character of the dictionary headword for entries that have a picture. This space makes the alignment inconsistent.

    • Roman Numeral Homographs displays the homograph numbers as roman numerals.

    • Section Subheaders collects the names of the writing systems and inserts them as the title of the section(s).

Some of these options affect the Table of Contents. You need to examine their effect with your data. If necessary, request technical support.

  1. Do the remaining steps in Set Defaults or Export Through Pathway.
