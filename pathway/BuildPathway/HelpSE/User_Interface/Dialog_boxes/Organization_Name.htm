Organization Name dialog box

The Organization Name dialog box appears when you click Other in the Select Your Organization dialog box.

In this case, the default copyright information is left incomplete. You will need to manually edit the Copyright Holder field (Publication Info tab) to include your organization name.

Do the following:

  1. Type the name of your organization, and then press OK.

    The name you typed is the selected organization in the Select Your Organization dialog box.

  2. Click OK to close the Select Your Organization dialog box, and open the Set Defaults dialog box.
