Open the Export Through Pathway dialog box

  1. In FieldWorks Language Explorer, do the following:

    • On the File menu, click Export.

    • In the Export dialog box, click Dictionary, Reversal Index Pathway (various options).

    • Click Export.

  2. If the Select Your Organization dialog box appears, select your organization, and then click OK.

The Export Through Pathway dialog box opens.
