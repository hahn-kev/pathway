Stylesheet pane

This pane is the right panel. It shows the name of the current stylesheet, and some of its details (properties).

  • Summary window: At the top of this pane is a non-editable window. It shows a summary of some selections set in the tabs below it.

  • Stylesheet tabs display information about the current stylesheet. For standard stylesheets, these settings are not editable. Add and customize a stylesheet as needed.

    • Info tab:

It stores information about the selected stylesheet. It also has the Available check box and Approved By field. See: User Interface terms - Info tab.

    • Properties tab:

It stores the display property selections for the current stylesheet. See: User Interface terms - Properties tab.

    • Preview tab:

It shows sample data that uses the stylesheet selected in the Stylesheets pane. Preview on the toolbar shows a larger view.

Tip

  • Publication metadata is stored in the Publication Info tab and Front Matter tab (Set Defaults and Export Through Pathway dialog boxes).
