Column Gap

When Paper is selected in the Navigation pane, the Properties tab shows the Column Gap field.

The Column Gap field shows the following:

  • For standard stylesheets, typically the absolute value of 12pt is displayed. You cannot change it.

  • For custom stylesheets, you could specify a relative value or an absolute value.

A red exclamation point () appears if you type a value that is not possible or outside of the range (6pt to 72pt) set by the program.

Note

  • The combination of the Column Gap selection and the Inside Margins and Outside Margins cannot exceed half the width of the page.

  • Column gap (also called Gutter width) is defined as the width of the space between columns.
