Maximum Image Width

With the EBook (epub) stylesheet (or a copy) selected in the Stylesheets pane, the Properties tab shows the Maximum Image Width field. The default width value is 600 pixels.

If you include images in a custom stylesheet, you can type a different value (in pixels).

  • If an image is narrower, the image is not changed.

  • If the image is wider, then the image is scaled down to the maximum default width value. The aspect ratio of the image is maintained.

Important

  • You may need to set a smaller default width value. For example, if you need to scroll or resize the reader window to see an entry that has a picture, the picture may be too large.
