Publisher

The Publisher field is in the Publication Info tab in both the Set Defaults and Export Through Pathway dialog boxes. It stores the name of the publisher of the data in the export file.

  • The selection in the Select Your Organization dialog box populate this field with approved content.

  • If you use the Organization Name dialog box, the organization name you entered appears in the field. You to type or paste the publisher name manually.
