TOC Level

With the EBook (epub) stylesheet (or a copy) selected in the Stylesheets pane, the Properties tab shows the TOC Levels (Table of Contents) field.

For a custom stylesheet, you can select one of these options:

Letter Only - the EBook reader TOC shows only a flat list of the first letters of the lexical entries in alphabetical order.

Letter and Entry - the EBook reader TOC shows a two-level hierarchial list:

  • the first level lists the first letters of the lexical entries in alphabetical order,

    • the second level lists all the lexical entries that begin with that letter.

Letter, Entry and Sense - the EBook reader TOC shows a three-level list:

  • the first level lists the first letters of the lexical entries in alphabetical order,

    • the second level lists the headwords of the lexical entries that begin with that letter,

      • the third level can list different sense-level content, depending on the configuration of the FieldWorks Language Explorer Dictionary view, and the content in each sense or subsense.

Tip

  • Sense-level content that appears in the third level can be grammatical information, sense number, gloss, or other. Factors that determine what is displayed include (not exhaustive) whether the entry has more than one sense, and if it does, are the parts of speech the same for each sense. Is a part of selected in each senses, and so on.

  • Use the Configure Dictionary View dialog box features in Language Explorer to configure the display of senses.
