Pictures

When Paper is selected in the Navigation pane, the Properties tab shows the Pictures field.

For custom stylesheets, you can choose one of the following:

  • Yes - to include any pictures in the output file.

  • No - to exclude any pictures from the output file.

Tip

Normally, exported pictures are positioned to the right in the dictionary entry.

You may be able to reposition pictures included in the output file, such as in LibreOffice (a destination). Some could use Prince to convert and edit the output file.
