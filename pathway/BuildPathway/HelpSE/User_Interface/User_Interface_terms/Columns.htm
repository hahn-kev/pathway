Columns

When Paper is selected in the Navigation pane, the Properties tab shows the Columns field.

For a custom stylesheet, you can select the number of columns that you want to use in the output file.

  • You can select 1 or 2 (columns).

    • If you select 2, then you need to specify a column gap.

You cannot add another option, such as a third column, using User Interface features. If three columns are required, either request this feature, or create a custom installation.
