Description (Publication)

This Description field is in the Publication Info tab in both the Set Defaults and Export Through Pathway dialog boxes. It stores the description of the publication. You need to enter or type the description of the publication manually.

Uses of the content stored in this field

  • It is included as metadata in export file.

  • For example, with EBook (.epub), the e-book reader will typically display the description along with the document’s other metadata and properties.

    • For instructions on how to view the properties of a document in your e-book reader, please refer to your e-book reader’s help.

Note

  • The Description field in the Info tab is for the selected stylesheet.
