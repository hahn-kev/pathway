No page breaks within letters

With the EBook (epub) stylesheet (or a copy) selected in the Stylesheets pane, the Properties tab shows the No page breaks within letters field. It uses a check box.

  • If selected (), each letter will be output as a single file.

Normally the file size is limited so that the e-book will display on e-readers with limited memory.

With this option selected, if the dictionary has a letter with a large number of entries, it will be stored as a single file and the e-reader will be required to manage the large file.

In this case, you need to test the output on the most limited e-reader used by your target audience.

  • If cleared (), the output will be multiple-smaller files.
