Line Spacing

When Paper is selected in the Navigation pane, the Properties tab shows the Line Spacing field.

In a custom stylesheet you can select the spacing between lines.

  • You can select point sizes of 11, 12, 13, 14, 16, 24, or No Change.

    • No Change means that the line spacing sizes specified by your content-providing program are not changed.

Important

  • Select a size that is appropriate for the Base Font Size you will use.

    • Standard stylesheets typically set the base font size one, two, or three points smaller than the line spacing. Stylesheets for drafts may specify a larger line spacing, so space is available for hand-written comments.

  • In typography, leading, pronounced ˈlɛdɪŋ, refers to the amount of added vertical spacing between lines of type. In Pathway, Line Spacing is used instead of leading.
