Pathway overview

The Pathway program has two parts, which you choose in the Pathway Installer:

Pathway Configuration Tool

The Pathway Configuration Tool is a stand-alone tool designed to help those with appropriate knowledge work with stylesheets.

The goals include the production of publishable language data from content-providing programs, such as SIL FieldWorks. The output destination can be paper or EBook, and mobile devices or web (future).

The Pathway Configuration Tool Help topics are listed under these overviews:

  • Installation overview

    • Known Issues overview

  • Concepts overview

    • Going Further overview

  • User Interface overview

    • User Interface terms overview

  • Tasks overview

Pathway in SIL FieldWorks

The Pathway Configuration Tool (http://pathway.sil.org/) enables the use of stylesheets to format output from SIL FieldWorks (http://fieldworks.sil.org/). The stylesheets do not change the formatting or layout of language data as it appears in SIL FieldWorks.

The related Help topics are:

  • Export Through Pathway dialog box

    • Preparation before exporting data

  • Select Layout dialog box
