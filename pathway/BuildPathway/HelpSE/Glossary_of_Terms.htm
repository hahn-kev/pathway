Glossary of Terms

Refer to Concepts in the Help file for information about terms or names of features you see in the UI (User-Interface).

To find a term in the glossary, click the letter that is the first letter in the word or phrase you want to look up.

G

gutter width - In publishing, gutter width is the width of the space between columns. In Pathway, this property is called Column Gap.

J

justified - In typography, when a line of text is justified, the spaces between words, and, to a lesser extent, between glyphs or letters, are stretched or compressed to make them align at both the left and right ends of each line of text. This contrasts to a ragged edge.

 

K

 

L

leading - In typography, leading, pronounced ˈlɛdɪŋ, refers to the amount of added vertical spacing between lines of type. In this User Interface, Line Spacing is used instead.

 

M

margin (page) - The blank space outside the printing area on a page.

Mobile - In Pathway Configuration Tool, mobile refers to all types of "smart devices", such as smartphones or PDAs for which you would need to configure a stylesheet.

 

N

 

O

 

Others - In Pathway Configuration Tool, Others is an optional place to store and access specialty custom stylesheets that do not fit neatly in Paper, Mobile, or Web.

P

point - In typography, a point is the smallest unit of measure. There are 72 points to the inch (1 point = 127⁄360 mm ≈ 0.353 mm, or 1/72 of an inch).

S

stylesheet (or style sheet) - a document that consists of settings that describe the layout and presentation to be used when displaying other documents. That is, stylesheets allow you to define how your pages will look. See: CSS.
See also: http://en.wikipedia.org/wiki/Cascading_Style_Sheets

 

T

type - as a verb, type means to use your keyboard to write; as a noun, type refers to a character or line of characters, which in typography encompasses type design, type glyphs, type arrangement, and so on.

typography - Typography comes from the Greek words τύπος typos "mark, figure" and γράφω grapho "I write". (http://en.wikipedia.org/wiki/Typography)

Z

Zip files - Zip files are single files (also known as archives) that contain one or more compressed files. In the Pathway Configuration Tool, the Send feature compresses the stylesheet in a zip file.

 

Symbols

New   - click to add a new stylesheet.

Save As - click to copy an existing stylesheet.

Delete   - click to delete current stylesheet.

Preview   - click to see a preview of the current stylesheet.

Defaults - click to open the Set Defaults dialog box.

Reset - click to delete custom stylesheets and Defaults (above) that you set to put Pathway back to its initial state.

Send - click to make a zip file of the current stylesheet and open your default email program so you can send it to someone.

Help - click to open this Help file, a pointer to the Student Manual, and open the About box.

Paper   - click to see stylesheets intended for publishing to paper.

Mobile   - click to see stylesheets intended for publishing to mobile or "smartdevices".

Web   - click to see stylesheets intended for publishing to the internet.
