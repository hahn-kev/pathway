calibre

calibre is a free and open source e-book library management application developed by users of e-books for users of e-books.

The Convert books feature in calibre allows you to convert books from EPUB format to other formats, such as to MOBI for use in Kindle products.

To use a Kindle Touch Previewer, for example, do the following:

  1. Export the data to the E-Book (epub) destination in the Export Through Pathway dialog box.

  2. Open the output file in calibre.

  3. Use the Convert books feature to convert the book to MOBI.

  4. Open the MOBI file in your Kindle Previewer.
