EBook (epub) or E-Book (.epub)

E-Book (.epub) is an open electronic book reader format that can be read by computers, smart phones, tablets and dedicated e-book reader software, such as calibre.

This format retains text styling and can contain embedded fonts for those writing systems that require it, but (unlike the .pdf format) will ”re-flow” text to fit the screen size of the device that is being used to view the document.

This format has the following settings (on the Properties tab of Stylesheet pane):

  • Layout

    • Font Size

    • Line height

    • Text alignment

    • Include Images

      • Maximum Image Width

    • TOC Level

  • Embedded Fonts

    • Embed Fonts in Document

    • Also Embed Font Variants

    • If font is missing

    • If font is non-SIL

    • Fallback font

Tip - Considerations for .epub documents

  • Embedding fonts in your document will allow non-western characters to display properly in e-book readers, but will also result in a larger file size. The file size can increase if you also choose to embed font variants (e.g., bold and italic versions of the fonts). You should consider whether embedded fonts are required for your document, or if you could publish the document without the embedded fonts.

  • When you embed a font in an *.epub document, you are redistributing it.

    • SIL fonts are freely redistributable, but this might not be the case for other fonts installed on your system. Before embedding a font, make sure you have the rights to redistribute it. The Embedded Fonts section of the .epub settings contains controls that can determine how the .epub export handles missing and non-SIL fonts.

  • The E-Book (.epub) destination has the capability to scale down large images down to a maximum width (this is done without distorting the image). The default maximum value is 600 pixels – you might need to test this value on a variety of input devices to ensure that your images render properly.

  • For Kindle, use the E-Book (.epub) destination. Then use calibre to covert EPUB to MOBI.
