Set up Web overview

To send dictionary data to Webonary, you export an XHTML file directly from FieldWorks Language Explorer. That is, you do not need to use Pathway. Currently, this may be problematic for large dictionaries.

Alternatively, you can pass the data through Pathway to web hosting site that you chose, such as a site set up on HostGator or WordPress.org. In this case, select Web in the Navigation pane to use or copy stylesheets related to web sites. Pathway processes the XHTML file so that there is separate import command for every 10 entries. This allows you to post a dictionary of any size.

  • In the Pathway Configuration Tool, the Properties tab has boxes where you enter details that are necessary to send your data to a website.

See: Set up web stylesheet.

Tip

  • There are many different web hosting services. Further, you may be on your own and will need set up the website yourself, or you may be part of an organization that has an existing website you can use. These considerations are beyond the scope of these Pathway User Helps.
