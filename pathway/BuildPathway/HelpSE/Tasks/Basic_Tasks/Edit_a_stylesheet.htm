Edit a stylesheet

  1. Display the appropriate list of stylesheets.

  2. In the Stylesheets pane, click the stylesheet you want to edit.

    • If needed, copy a standard stylesheet or custom stylesheet, and then edit the copy.

  3. In the Stylesheet pane, click the Info tab, and then do any of the following:

    • Type the name of the stylesheet.

    • Type the description of the stylesheet.

    • If you want to make the stylesheet available, select () Available check box.

    • Type a comment (optional).

  4. For a custom stylesheet, click the Properties tab, and then do the following:

    • Type a new value or make a different selection in each field that you need to change.
