Archive data through Pathway

You can export data through Pathway to any available destination. Pathway generates an equivalent <name>.ramp file which you can import into RAMP (SIL). It is in the same file folder as the other output file.

  • RAMP is only available for download in Insite (SIL only) — in Insite, search for REAP (Repository for Electronic Archiving and Publishing).

There you can also find the RAMP Users' Manual, other documentation and more.

  • See Also: RAMP Metadata.
