Sending stylesheets

You can use Pathway features to put a set of stylesheets in a compressed zip file and email it. You could do this to keep a recipient's set of stylesheets the same as yours. When you do the Send steps below, all the stylesheets in the displayed list are put in the zip file.

Important

  • You need to have a default mail client properly installed. Otherwise, the Pathway Configuration Tool will stop working as it tries to find the mail client.

Send

To send stylesheets to someone by email, do the following steps:

  1. Display the appropriate list of stylesheets.

  2. In the Stylesheets pane, click the stylesheet you want to send.

  3. On the toolbar, click Send, or press the shortcut keys Alt +E.

    Pathway does the following actions:

    • Creates and exports a zip file to your computer's hard drive.

    • Opens your default email program.

  4. In your email program, do the following:

    • Attach the zip file to the email.

    • Type the recipients email address, and any additional comments the body of the email.

    • Send the email.

Receive

As a recipient of an email with a zip file, do the following to put the stylesheets where Pathway can find them:

  1. Extract the zip contents to a folder on your hard drive.

  2. Copy the two settings files, DictionaryStyleSettings.xml and styleSettings.xsd, to the path

    Windows XP:

    • C:\Documents and Settings\All Users\Application Data\SIL\Pathway\Dictionary

    Windows Vista or Windows 7 and 8:

    • C:\Users\All Users\Application Data\SIL\Pathway\Dictionary

    Ubuntu 12.04 (precise):

    • ~/.local/share/SIL/Pathway/Dictionary.

  3. Copy the all css files to the path

    Windows XP:

    • C:\Documents and Settings\All Users\Application Data\SIL\Pathway\Dictionary

    Windows Vista or Windows 7 and 8:

    • C:\Users\All Users\Application Data\SIL\Pathway\Dictionary

    Ubuntu 12.04 (precise):

    • ~/.local/share/SIL/Pathway/Dictionary.
