Display the appropriate list of stylesheets

Before you work with a stylesheet (edit, copy, send, add, set default values, and so on), make sure the appropriate list of stylesheets is displayed in the Stylesheets pane. Otherwise, you could inadvertently work with the wrong stylesheet.

  • In the Navigation pane, click one of the following:

    • Paper

    • Mobile

    • Web

    • Others
